# PTF Studio Beta 0.9.5.5

## Interface refinements

- All coloured action buttons now use white text and white SVG icons.
- Hover, pressed and disabled states retain the white-label treatment.
- XMB category and icon-title text now uses a softer blurred drop shadow rather than a hard duplicated offset.
- The Asset Maker wallpaper XMB overlay uses the same softened title-shadow rendering.

All PTF import, export, Asset Maker, PSP Go and focus-generation features from Beta 0.9.5.4 remain unchanged.
