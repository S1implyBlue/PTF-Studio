# PTF Studio Beta 0.9.3

Beta 0.9.3 expands PTF Studio from an individual-asset editor into a broader theme-production workflow.

## Added

- PSP model profiles for PSP-1000, PSP-2000, PSP-3000, PSP Go, PSP Street, and Universal projects.
- Compatibility badges and profile-aware live preview navigation.
- Automatic creation of missing model-specific and fallback slots.
- Bulk icon-folder import with numeric and descriptive filename matching.
- Focus-glow generator with colour, opacity, blur, padding, and scope controls.
- Theme analyser covering dimensions, palette use, compression, packed size, and estimated final PTF size.
- CSV export for analyser results.
- Theme variant builder with colour metadata and optional icon tinting.
- ZIP export for multiple PTF variants.
- Release ZIP generator containing the PTF, preview image, icon sheet, README, credits, and manifest.
- Direct Default/Fallback asset filtering and editing.

## Compatibility retained

- Legacy RLZ/LZR import and preservation.
- Modern Deflate and stored-resource import/export.
- Existing XMB preview, replacement, bulk first-level focus, metadata, and export features.

## Deliberately omitted

Screenshot comparison mode was not included at the creator's request.
