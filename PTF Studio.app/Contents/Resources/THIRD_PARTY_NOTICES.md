# Third-Party Notices

PTF Studio is dependency-free at runtime and does not bundle Sony firmware, fonts, or theme assets beyond the user-provided sample theme.

The legacy LZR/RLZ decoder is a JavaScript implementation of the publicly documented `sceLzrDecompress` stream semantics. Compatibility research was cross-checked against the open-source PlayFieldPortal theme-kit implementation and PSP homebrew format documentation.

Poppins is requested from Google Fonts for the application interface and is not bundled in the portable package. Poppins is distributed under the SIL Open Font License.
