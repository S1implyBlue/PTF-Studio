# PTF Studio Beta 0.9.5.1

## Fixed: Asset Maker canvas proportions

- Fixed the Asset Maker drawing surface being stretched to the modal panel dimensions.
- Square presets now display as true squares: 16 × 16, 32 × 32, 48 × 48, and 64 × 64.
- Category icons, preview images, and wallpapers retain their correct native aspect ratios.
- Added automatic canvas backing-store synchronization for browser zoom, Retina/HiDPI displays, modal resizing, and window resizing.
- Corrected pointer-to-canvas coordinate mapping after responsive resizing.
- Transparent checkerboards, grid lines, selection outlines, imported artwork, and the indexed-colour preview now share identical proportions.
- No PTF import, editing, compression, metadata, or export behavior was changed.

## Compatibility

All Beta 0.9.5 Asset Maker features and all earlier PTF Studio features remain available.
