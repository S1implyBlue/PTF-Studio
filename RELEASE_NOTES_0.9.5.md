# PTF Studio Beta 0.9.5

## Added: Asset Maker

- Layer-based maker for every standard PSP PTF image size.
- Wallpaper, preview image, category icon, first-level body/focus, second-level body/focus, and preview-icon presets.
- Artwork import with transparent-canvas preservation.
- Rectangle, rounded rectangle, circle, and line tools.
- Move, resize, rotate, opacity, visibility, layer ordering, duplication, deletion, undo, and redo.
- Basic gradients, strokes, tint, brightness, contrast, and saturation.
- PSP-conscious baked outline, solid shadow, and low-radius glow controls.
- Indexed 256-colour preview for PSP icon assets.
- Wallpaper XMB readability overlay.
- Matching focus generation from body icons.
- Direct application to compatible theme slots and direct XMB preview.
- PNG export from the maker canvas.

## Not included yet

The standard/sample icon library is deferred to the next release while the redistributable asset pack is being prepared.

## Compatibility

All Beta 0.9.4.3 PTF import, RLZ/LZR support, image downscaling, metadata, colour selection, preview, and export behaviour remains available.
